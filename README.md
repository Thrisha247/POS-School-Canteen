School Canteen POS System

Overview
The School Canteen POS System is a Java-based Point of Sale application designed to facilitate the management of a school canteen. The system provides functionality for user authentication, menu management, order placement, and calculation of total bills. All data is stored and retrieved using a MySQL database.  

 Features
- Secure login for administrators and students  
- Add, update, and view canteen menu items  
- Place orders and calculate total price  
- Intuitive Java Swing graphical user interface  
- Persistent storage using MySQL  

 Technology Stack
- *Programming Language:* Java  
- *GUI Framework:* Java Swing  
- *Database:* MySQL  
- *Database Connectivity:* JDBC  

 Database Structure
- **canteen_menu** – Stores dishes available each day  
- **users** – Stores user credentials
